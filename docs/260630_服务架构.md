## 3. 各组件职责与 HA

| 组件 | 进程 | 状态 | 挂了怎么办 |
|------|------|------|------------|
| **MCP Server** | 独立 | 无状态 | bootstrap 跳过该域，`health=degraded` |
| **harness-core** | 库 | — | 随服务发布版本 |
| **trace / 8D Worker** | 独立 | 无状态 | Orchestrator 重试 |
| **rca_agent** | 独立 | LangGraph checkpoint | Redis Saver；HITL thread_id |
| **orchestrator** | 独立 | Context 内存/Redis | Redis Session |
| **capability-registry** | 独立 | 种子只读 | 多副本无冲突 |
