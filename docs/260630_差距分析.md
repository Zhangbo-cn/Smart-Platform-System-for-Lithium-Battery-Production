# 架构缺口与文档一致性审阅

> **版本**：v1.1 | 2026-06-30  
> **范围**：历史审阅记录  
> **用途**：演进优先级参考；**实现进度以 [IMPLEMENTATION_STATUS.md](./260630_实现状态.md) 为准**

---

## 1. 总体结论

| 维度 | 评估 | 说明 |
|------|------|------|
| 架构愿景 | **强** | A2A + MCP + LangGraph RCA + Safety 门闩，贴合锂电制造 |
| Spec 完整度 | **中上** | 主架构、协议、RCA、Playbook 已齐 |
| 文档一致性 | **已整治** | Supervisor→Planner+Router；Playbook 统一；能耗 Agent 移出主清单 |
| 可运行程度 | **中高** | Gateway + Orchestrator + Registry + trace/8d Worker 竖切可跑；Planner MVP |
| 可背书模块 | **高** | RCA + 四域 MCP + FMEA + Harness |

**一句话**：设计文档可支撑「13 Agent 中台」叙事；**可演示代码**仍以 RCA 单服务为主。下一竖切：**AgentCard → Router → trace → investigate 剧本**。

## 2. 架构缺失（按优先级）

### P0 — 平台成立阻断项

| 缺失 | Spec | 代码 | 影响 |
|------|------|------|------|
| Client Gateway | ✅ | ✅ | `services/client-gateway/` |
| Planner | ✅ | ⚠️ MVP | `services/planner-agent/` |
| Orchestrator / Registry | ✅ | ✅ | `services/orchestrator/` · `capability-registry/` |
| PlatformContext | ✅ | ⚠️ Router 内存 | 生产待 Redis |
| Safety + plc MCP | ✅ | ❌ | 停线门闩空缺 |
| 追溯 / 分诊 | ✅ | ✅ trace Worker + triage_stub | investigate 竖切 |
| QMS 写回 | ✅ | ✅ | report-agent + qms MCP |
| L5 门户 | ✅ | ✅ | Gateway @ 8010 |
| 多工厂 tenant | ✅ | ❌ | 未隔离 |

### P1 — 生产运营阻断项

| 缺失 | 说明 |
|------|------|
| A2A `tasks/send` 薄适配 | RCA 已有 `/a2a/v1/tasks/send` |
| Registry 健康检查 | AgentCard `enabled` 无探活 |
| MCP 熔断全链路 | 规划有，未全接入 |
| Eval / Golden Set 闭环 | F1、nDCG 等未完整实现 |
| SSE 任务推送 | Client 能力未协议化 |
| PlatformContext JSON Schema | 字段矩阵有，无 schema 文件 |

### P2 — 扩展项（主架构已移出或后置）

| 项 | 决策 |
|----|------|
| **能耗调度 Agent** | **不纳入 13 Agent 主清单**；能效 KPI 经 L5 驾驶舱 + EMS 只读；若二期需要再单独立项 P3 |
| wms / eam / qms MCP | 九域 Spec 有，代码仅四域 |
| 业务实例图谱（批次-物料-设备） | Neo4j 偏 FMEA；追溯图谱 Spec 待补 |
| 客诉 / COPQ Agent | 未入当前 Spec，按需扩展 |
| 模型治理 / 成本配额 | P1 规划，无细则 |

## 5. 仍须注意的口径风险

| 来源 | 风险 | 建议 |
|------|------|------|
| `简历_锂电智能化平台.md` 成果表 | 追溯 &lt;5min、分诊 91% 等无全量代码支撑 | 标注「设计目标 / 部分验证」或面试时限定 RCA 指标 |
| `ARCHITECTURE_A2A_MCP` §2「去中心化」 | 易误解为 Agent 网状互连 | 已改为 Router 居中；能力解耦 ≠ Agent 直连 |
| `client-agent` `enabled: true` | 实际未实现 | 实现前改为 `false` 或注明 Spec-only |
