## 顶层目录

```
Battery_Agent_DS_Aug/
├── packages/platform-contracts/
├── packages/harness-core/
├── services/
│   ├── client-gateway/           # 用户门户（Gateway，无 LLM）
│   ├── planner/                  # 任务规划（Agent，P1）
│   ├── orchestrator/             # Playbook 编排（无 LLM）
│   ├── capability-registry/      # 能力服务登记
│   ├── a2a_server/
│   │   ├── trace_worker/
│   │   ├── report-agent/
│   │   └── rca-agent/            # junction → ../Battery_Agent_DS

├── deploy/docker-compose.platform.yml

```
